
## Clone

Install my project using clone

```bash
  git clone https://github.com/Mssskrishna/OdinCodes.git
  cd BlogApp
```
## Install 

Install dependancies and Dev-dependancies

```bash
  cd backend
  npm install
```
```bash
  cd frontend
  npm install
```
```bash
  cd userfrontend
  npm install
```
## See results

Run following commands to start the backend server in backend folder

```bash
  cd backend
  npm run devStart
```
```bash
  cd userfrontend
  npm run dev
